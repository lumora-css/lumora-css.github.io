title: Typography

Documentation and examples for typography, including global settings, headings, body text, lists, and more.

### Headings
All HTML headings, `<h1>` through `<h6>`, are available. `.h1` through `.h6` classes are also available, for when you want to match the font styling of a heading but cannot use the associated HTML element.

<div class="preview-block">
  <h1>h1. Lumora heading</h1>
  <h2>h2. Lumora heading</h2>
  <h3>h3. Lumora heading</h3>
  <h4>h4. Lumora heading</h4>
  <h5>h5. Lumora heading</h5>
  <h6>h6. Lumora heading</h6>
</div>

```html
<h1>h1. Lumora heading</h1>
<h2>h2. Lumora heading</h2>
<h3>h3. Lumora heading</h3>
<h4>h4. Lumora heading</h4>
<h5>h5. Lumora heading</h5>
<h6>h6. Lumora heading</h6>
```

### Display Headings
When you need a heading to stand out, consider using a display heading—a larger, slightly more opinionated heading style.

<div class="preview-block">
  <h1 class="display-1">Display 1</h1>
  <h1 class="display-4">Display 4</h1>
</div>

```html
<h1 class="display-1">Display 1</h1>
<h1 class="display-4">Display 4</h1>
```

### Lead
Make a paragraph stand out by adding `.lead`.

<div class="preview-block">
  <p class="lead">This is a lead paragraph. It stands out from regular paragraphs.</p>
</div>

```html
<p class="lead">This is a lead paragraph. It stands out from regular paragraphs.</p>
```

### Inline text elements
Styling for common inline HTML5 elements.

<div class="preview-block">
  <p>You can use the mark tag to <mark>highlight</mark> text.</p>
  <p><small>This line of text is meant to be treated as fine print.</small></p>
</div>

```html
<p>You can use the mark tag to <mark>highlight</mark> text.</p>
<p><small>This line of text is meant to be treated as fine print.</small></p>
```
