Lumora provides an elegant reboot, an evolution of Normalize.css, providing a simple and elegant baseline to build upon.

### Approach
Reboot provides opinionated resets to various HTML elements so that component styles build on a predictable baseline.

<div class="preview-block">
  <h4>Headings, Paragraphs, and Lists</h4>
  <p>All headings have margins removed from the top and standardized on the bottom. Paragraphs also have standardized bottom margins.</p>
  <ul>
    <li>Lists have their top margins reset</li>
    <li>Nested lists have no bottom margin</li>
  </ul>
  <hr>
  <p>Blockquotes have consistent spacing, and horizontal rules (`hr`) have predictable styling.</p>
</div>
