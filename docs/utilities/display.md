---
title: Display
---

Quickly and responsively toggle the display value of components and more with our display utilities. Includes support for some of the more common values.

<div class="preview-block">
  <div class="d-inline p-2 bg-primary text-white">d-inline</div>
  <div class="d-inline p-2 bg-dark text-white">d-inline</div>
  
  <span class="d-block p-2 bg-primary text-white mt-3">d-block</span>
  <span class="d-block p-2 bg-dark text-white mt-1">d-block</span>
</div>

```html
<div class="d-inline p-2 bg-primary text-white">d-inline</div>
<div class="d-inline p-2 bg-dark text-white">d-inline</div>

<span class="d-block p-2 bg-primary text-white mt-3">d-block</span>
<span class="d-block p-2 bg-dark text-white mt-1">d-block</span>
```

### Supported values
- `.d-none`
- `.d-inline`
- `.d-inline-block`
- `.d-block`
- `.d-flex`
- `.d-inline-flex`
