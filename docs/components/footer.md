# Footer

The Footer component provides a structured section for the bottom of your pages, typically containing links, copyright information, and social media icons.

## Basic Usage

```html
<footer class="footer">
  <div class="container">
    <div class="footer-content">
      <div class="footer-section">
        <h4 class="footer-title">Lumora</h4>
        <p class="footer-text">A beautiful UI CSS framework.</p>
      </div>
      <div class="footer-section">
        <h4 class="footer-title">Links</h4>
        <ul class="footer-links">
          <li><a href="#" class="footer-link">Home</a></li>
          <li><a href="#" class="footer-link">Features</a></li>
        </ul>
      </div>
    </div>
    <div class="footer-bottom">
      <p class="footer-text">&copy; 2026 Lumora. All rights reserved.</p>
      <div class="footer-social">
        <a href="#" class="footer-social-link">Twitter</a>
        <a href="#" class="footer-social-link">GitHub</a>
      </div>
    </div>
  </div>
</footer>
```
